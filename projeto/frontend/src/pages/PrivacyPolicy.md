# Política de Privacidade - MedSync

**Última Atualização:** 26 de Fevereiro de 2026

## 1. Coleta de Dados
O MedSync coleta as seguintes informações dos usuários (médicos e pacientes):
- **Dados Pessoais:** Nome, CPF, E-mail, Telefone, CRM (para médicos).
- **Dados Sensíveis:** Histórico médico, prontuários, exames laboratoriais e laudos. Este dado sensível é criptografado utilizando AES-256-GCM antes de ser armazenado no banco de dados.
- **Dados de Navegação:** Cookies de sessão essenciais.

## 2. Uso dos Dados
Os dados coletados têm a finalidade exclusiva de:
- Autenticar o acesso de médicos e pacientes aos prontuários médicos.
- Prestar serviços de telemedicina e acompanhamento contínuo.
- Emitir logs de auditoria de segurança (conforme obrigações descritas na legislação).

## 3. Compartilhamento de Dados
**O MedSync NÃO vende, aluga ou cede dados pessoais ou sensíveis a terceiros.**
Os dados podem ser compartilhados unicamente em caso de:
- Ordem judicial.
- Transferência de dados a pedido explícito do paciente para outro profissional ou instituição de saúde (Portabilidade).

## 4. Retenção e Anonimização de Dados
Após 12 meses (1 ano) de inatividade, os dados pessoais (nome, email, documento) do usuário são **Anonimizados Automaticamente**, preservando o histórico clínico apenas para fins de pesquisas demográficas sanitárias sem identificação do indivíduo.

## 5. Direitos do Titular da Informação (LGPD)
Você tem o direito de:
1. **Confirmação:** Confirmar a existência de tratamento dos seus dados.
2. **Acesso e Correção:** Acessar seus prontuários e corrigir dados incompletos ou inexatos.
3. **Portabilidade:** Ao solicitar pelo seu painel, você receberá um arquivo `.JSON` ou `.CSV` contendo todos os seus dados.
4. **Anonimização e Bloqueio:** Para dados desnecessários ou excessivos.
5. **Direito ao Esquecimento:** Você pode solicitar a eliminação dos seus dados a qualquer momento. Em 30 dias a exclusão lógica é feita e você não terá mais acesso de login. Em 180 dias os dados são apagados de forma permanente dos nossos servidores (se não conflitarem com as leis de guarda de prontuário médico vigentes).

## 6. Contato com o DPO (Data Protection Officer)
Se houver qualquer dúvida ou solicitação, contate nosso Encarregado de Dados em: `dpo@medsync.com.br`.
